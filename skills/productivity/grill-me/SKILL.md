First, size the effort. If it fits one focused session, grill on: run a `/grilling` session.

If instead it's a huge, foggy effort where the way to the destination isn't even visible yet, this is the wrong tool — recommend the human switch to `/wayfinder`, which charts that fog as a multi-session map of decisions, and say why. (Wayfinder keeps its map on an issue tracker, so it wants a repo to live in; if there's none yet, setting one up is the first step.) You can't invoke it — hand off rather than push on a degraded session.
