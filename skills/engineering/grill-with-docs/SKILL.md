First, size the effort. If the whole thing — the sharpening, the spec, and the design overview — can plausibly fit one focused session, grill on: run a `/grilling` session, using the `/domain-modeling` skill.

If instead it's a huge, foggy effort where the way to the destination isn't even visible yet — a greenfield build, a sprawling multi-feature push — this is the wrong tool. **Stop and recommend the human switch to `/wayfinder`**, which charts that fog as a multi-session map of decisions, and say why. You can't invoke it — it's user-invoked, like this skill — so hand off rather than push on a degraded session. When in doubt, or when it clearly fits one session, grill.
